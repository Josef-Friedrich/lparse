# lparse

Parse and scan macro arguments in `Lua` on `LuaTeX` using a `xparse`
like argument specification and never get dirty with `#1`, `#2`, `#3`,
`\luaescapestring`, `\luatexluaescapestring`, `\unexpanded` again.
